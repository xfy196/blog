## Scaffolding Your First Blog Project

With NPM:

```bash
$ npm create 小小荧@latest
```

With Yarn:

```bash
$ yarn create 小小荧
```

With PNPM:

```bash
$ pnpm create 小小荧
```

Then follow the prompts!

You can also directly specify the project name you want to use via additional command line options. For example, to scaffold a my-first-blog, run:

```bash
# npm
npm create 小小荧@latest my-first-blog

# yarn
yarn create 小小荧 my-first-blog

# pnpm
pnpm create 小小荧 my-first-blog
```
