# 格式转换

## Convertio

#### **🏆优点**

+   速度快、格式多、免费免登录
+   应对生活大多数场景是够用的

#### **🥷缺点**

+   超过100m的文件就要登录付费了

## 123APPs

#### **🏆优点**

+   这个免费的文件上限是10G，完完全全够用
+   功能多、除了转换器格式还有视频、音频、PDF处理工具

#### **🥷缺点**

+   广告稍微稍微有点多
+   速度略慢、有时候一些格式转换不成功

#### [点击跳转](https://123apps.com/cn/#)
