# 如何一键清除文件目录下所有的node_modules

快速删除目录下的`node_modules`，下面附上`windows`和`mac`的脚本指令

**windows脚本**
```sh
FOR /d /r . %d in (node_modules) DO @IF EXIST "%d" rm -rf "%d"
```
**mac脚本**
```sh
find . -name "node_modules" -type d -prune -exec rm -rf '{}' +
```
